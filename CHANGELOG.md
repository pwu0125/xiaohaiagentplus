# 小海Agent Plus 更新日志

## V7.0.0 (2026-06-15) — 稳定版本

### 架构升级
- 双流程独立设计：材料分析 + 模拟投决会两条独立流程
- AI-product 材料分析功能完整接入
- 知识库统一归档：ai-product 和 committee 两种分析结果

### 新增功能

#### 材料分析 Tab
- 文件上传支持 PDF / Excel / CSV / TXT
- 三态切换：上传 → 流式进度 → 报告展示
- AI-product 格式完整报告（12个章节）
- 宏观/行业周期图表（Recharts）
- 三情景对比（乐观/基准/悲观）
- 风险矩阵 + 行动建议

#### 模拟投决会
- 秘书Agent → 9部门并行 → 24位大师 → 决策者
- 流式SSE事件驱动，实时展示进度
- 完整的投决会报告（结论/评级/评分/风险/建议）

#### 多Agent对话
- 12位投资大师逐条流式回复
- 每位大师3条特色回复，根据话题动态选择
- 实时讨论进度指示

#### 知识库
- 9条Mock项目笔记（材料分析+投决会+研究笔记）
- 标签筛选 + 关键词搜索
- 类型徽章（材料分析/投决会/笔记）
- 内容展开/折叠
